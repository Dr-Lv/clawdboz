# 定时任务 Skill

## 描述
创建和管理定时任务，在指定时间自动执行并向用户发送结果。

## 功能

### 1. 创建定时任务
用户可以通过自然语言设置定时任务：
- "设置一个定时任务，明天上午9点提醒我开会"
- "10分钟后帮我查一下今天的天气"
- "今天下午3点总结这个群的所有消息"

### 2. 列出定时任务
- "列出所有定时任务"
- "查看我的定时任务"

### 3. 取消定时任务
- "取消定时任务 #1"
- "删除任务 #2"

## 支持的时间格式

### 相对时间
- "X分钟后" - 如 "10分钟后"
- "X小时后" - 如 "1小时后"
- "X天后" - 如 "2天后"

### 绝对时间
- "明天上午9点"
- "今天下午3点"
- "YYYY-MM-DD HH:MM" - 如 "2024-01-15 09:00"

## 工作原理

1. 用户创建定时任务后，任务会被保存到本地文件
2. 调度器会在指定时间触发任务
3. 任务执行时，会调用 ACP 处理用户的指令
4. 执行结果通过飞书消息发送给用户
5. 即使 Bot 重启，未执行的任务也会自动恢复

## 注意事项

1. 任务执行时，Bot 必须处于运行状态
2. 已过期的任务会被自动清理
3. 每个聊天独立管理自己的任务
4. 任务描述越详细，执行结果越好
